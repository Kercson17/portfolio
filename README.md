# Kercson G. Didal - Portfolio 🚀

A professional, minimal, and formal personal portfolio website built with [Next.js](https://nextjs.org/), [Tailwind CSS](https://tailwindcss.com/), and [TypeScript](https://www.typescriptlang.org/).

Inspired by clean, high-contrast design principles, this portfolio showcases my technical expertise, academic background, and selected projects.

## 🌟 Features

-   **Minimalist Design**: High-contrast, clean typography (Inter font) for a professional look.
-   **Responsive Layout**: Fully optimized for desktop, tablet, and mobile devices.
-   **Project Showcase**: Highlighted selected works with visual previews and tech stack tags.
-   **UI/UX Samples**: Dedicated section for design and creative portfolio samples.
-   **Technical Stack**: Categorized overview of frontend, backend, database, and tool expertise.
-   **Education Timeline**: Clear presentation of academic history and achievements.
-   **Direct Contact**: Easy access to email, phone, and professional links.
-   **CV Access**: Integrated "View" and "Download" options for my professional CV.

## 🛠️ Tech Stack

-   **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
-   **Deployment**: [Vercel](https://vercel.com/) (Recommended)

## 🚀 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) to see the portfolio!
